O que é o Praxis?

Praxis é uma abordagem orientada a metadados para acelerar o desenvolvimento de backends e experiências de UI. Através de anotações e convenções, geramos contratos OpenAPI mais ricos, schemas direcionados e endpoints utilitários.

  • UI Schemas para formular rapidamente telas/fluxos.
  • Filtros tipados e endpoints de options para selects/autocomplete.
  • Enriquecimento de documentação e padronização de padrões REST.

Praxis Cockpit

Explore este host como um mapa vivo de domínio. O cockpit é servido automaticamente pelo starter e mostra áreas de negócio, recursos, endpoints, filtros, tabelas, formulários, gráficos, actions, prontidão semântica e relações entre recursos.

  • URL: /praxis/cockpit
  • Leitura executiva do que o backend publica para humanos, runtimes e IA.
  • Sem HTML próprio neste host: os assets vêm do praxis-metadata-starter.

Sobre este Quickstart

Este projeto demonstra uma API com entidades e filtros exemplo (HR), plugada no Starter de Metadata, pronta para deploy em plataformas como Render.

  • Perfis dev e prod.
  • server.port a partir da env PORT.
  • Swagger UI público; APIs protegidas com cookie de sessão (JWT).

Como começar

Construa localmente e abra a documentação interativa.

  • Build: ./mvnw -B -DskipTests package
  • Rodar: java -jar target/praxis-api-quickstart-2.0.0-rc.9.jar
  • Docs: /swagger-ui/index.html

🦸‍♂️ Universo dos Heróis

Domínio temático usado para demonstrar CRUDs, relacionamentos e regras de negócio em um contexto lúdico. Plataforma: Spring Boot 3 (Java 21) + PostgreSQL 17.

Este domínio fornece dados e endpoints para demonstrar o ecossistema Praxis (Metadata Starter + UI). Veja detalhes no README e no documento de banco.
